VGMS Expansion Pack: Video Game Visual Styles

PoolParty Reference Number: 1DF197A6-3D38-0001-999A-1D2815D01AA1

  • Description

    This controlled vocabulary is a proposed expansion to the UW GAMER Group's Video Game Metadata Schema (VGMS) focusing on the visual styles of video games.

    While games are an overwhelmingly visual medium, the current descriptive practices for video game visual styles in the VGMS are very limited and make cataloging and retrieving games more difficult. These terms are meant to create a reliable index for gamers and non-gamers alike as they search for, and choose, games.

    This vocabulary is a prototype, and if accepted into the VGMS will continue to be expanded and edited as video games evolve as a medium.

    This is the creators' final project for LIS 536, Metadata for Interactive Media, in the MLIS program at the University of Washington.
